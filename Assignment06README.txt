#Python Traceroute Utility

#Objective
A Python implementation of a basic traceroute tool using the "scapy" library.

#Requirements
	Python 3.x
	Scapy library

#Usage

	python tracert google.com -m20 -t3

#Test Cases

Run the program with domain 'google.com'
	python tracert google.com

Run the program with domain 'Athens.edu'
	python tracert Athens.edu

Run the program with domain 'rapidcare.us'
	python tracert 'rapidcare.us'
